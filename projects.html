Pawdli - Pet Therapy & Adoption Platform

Cross-Platform Mobile Application for Pet Care

Pawdli is a comprehensive cross-platform Flutter application designed for pet therapy, adoption, and social connection features. The app provides a complete ecosystem for pet lovers, offering therapeutic sessions with pets, adoption services, and a buddy system to connect pet owners and enthusiasts.

The application features dynamic pet profiles with image galleries, session booking for pet therapy, adoption slot management, and interactive live sessions. Built with Firebase for robust backend support and Agora for real-time video communication, Pawdli delivers a seamless and engaging experience for users looking to connect with pets in meaningful ways.

Key Features

  • Pet Therapy Booking system with session scheduling
  • "Goodbye Buddy" feature for emotional farewell support
  • "Make a Buddy" social connection for pet owners
  • Pet adoption workflows with slot booking
  • Dynamic pet profiles with image galleries
  • Firebase authentication and cloud storage integration
  • Agora integration for live interactive pet therapy sessions
  • Real-time updates and notifications
  • Smooth navigation and responsive UI across all features

Technologies Used

Flutter Dart Firebase Agora SDK Cloud Storage REST API GetX FCM Real-time Database

College Management App

Cross-Platform Academic Management Solution

College Management App is a comprehensive cross-platform mobile application built using Flutter to streamline academic activities for both faculty and students. The app serves as a central hub for managing all aspects of college life, from attendance tracking to fee payments, making academic administration more efficient and accessible.

The application features role-based access control, allowing faculty members to upload attendance, publish results, share announcements, and manage student records, while students can track their attendance, view exam scores, access the academic calendar, make fee payments, and receive real-time notifications. Built with a focus on user experience and backend integration, the app delivers a seamless experience across both Android and iOS platforms.

Key Features

  • Real-time push notifications for announcements and updates
  • Academic calendar with events and exam schedules
  • Online fee payment integration with transaction history
  • Attendance tracking for students and marking system for faculty
  • Exam scores and result publication system
  • Faculty dashboard for uploading attendance and publishing results
  • Announcement and notice board for college-wide communication
  • Student record management for faculty
  • Role-based access control for students and faculty
  • Responsive UI/UX design with intuitive navigation

Technologies Used

Flutter Dart GetX REST API Firebase FCM Payment Gateway Role-Based Auth Cloud Storage
